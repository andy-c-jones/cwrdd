# Subscription Model and Pricing

## Overview

cwrdd is designed to be sustainable while keeping core features free. Most functionality is available without cost, but advanced organizational features—particularly public forums—require subscriptions to support the infrastructure and moderation costs.

## Core Principles

- **Free core functionality**: Individual users and basic organizations operate for free
- **Sustainable platform**: Subscriptions fund infrastructure and development
- **Value alignment**: Paid features support community building and democratic participation
- **Transparent pricing**: Clear tiers with no hidden fees
- **Fair limits**: Free tiers are generous, paid tiers remove friction

## Free Features

### For Individual Users

**Always Free:**
- Account creation and management
- Public and private profiles
- Direct messaging (end-to-end encrypted)
- Create and attend unlimited events
- Join and participate in organizations
- Access public channels and forums
- Create personal events
- Standard support

**No Limitations:**
- Number of contacts
- Number of messages
- Event creation
- Profile customization
- Organization memberships

### For Organizations (Free Tier)

**Included:**
- Organization profile creation
- Unlimited members
- Unlimited public chat channels
- Limited private channels
- Limited private forums (boards and threads)
- Event creation and management
- Basic analytics
- Standard support

**Free Tier Limits:**
- Private channels: Limited quantity per organization
- Private forums: Limited boards and total threads
- No public forums (paid feature)
- Message history retention: Limited timeframe
- No advanced moderation tools
- No custom branding

## Paid Features

### Public Forums (Organization Subscription Required)

**Why Paid:**
- Significant infrastructure costs (storage, indexing, search)
- Moderation tools development and maintenance
- Public forums are high-value, community-building features
- Revenue enables platform sustainability
- Comparable platforms charge for forum hosting

**What's Included:**
- Creation of public forums
- Organized boards and threads
- Rich moderation tools
- Analytics and insights
- Custom styling options (higher tiers)
- Priority support

### Subscription Tiers (Organizations)

Organizations choose a tier based on their forum and community needs.

#### Tier Structure

**Free Tier:**
- No public forums
- Limited private channels
- Limited private forum capacity (boards and threads)
- Basic features only

**Paid Tiers:**
Multiple subscription levels with increasing capabilities:
- Number of forum boards
- Advanced moderation tools
- Analytics and reporting
- Custom branding options
- Increased private channel capacity
- Unlimited private forum capacity
- Priority support
- API access (higher tiers)

**Pricing Philosophy:**
- Entry tier affordable for small organizations
- Mid-tier for growing communities
- High tier for large, active communities
- Annual billing discounts available

### Private Forum Capacity

**Free Tier:**
- Limited number of boards
- Limited total threads across all boards
- Adequate for small, private discussions
- Easy to start without payment

**Paid Tier:**
- Increased or unlimited boards
- Increased or unlimited threads
- Suitable for large, active private communities
- More affordable than public forum tiers

**Rationale:**
- Private forums have less infrastructure burden (not indexed, not searchable publicly)
- Generous free tier encourages adoption
- Paid tier removes constraints for active communities
- Revenue helps sustain platform

## Pricing Transparency

### What Organizations Get

**Clear Value Proposition:**
- Public forums: Community hub for broader audience
- Private capacity: Room to grow without artificial limits
- Advanced tools: Better moderation and management
- Analytics: Understand community engagement
- Support: Priority assistance when needed

### What Organizations Don't Pay For

**Unlimited on All Tiers:**
- Public chat channels (always free)
- Organization members (no per-seat pricing)
- Event creation and promotion
- Basic moderation tools
- Standard support

## Billing and Payment

### Payment Methods

- Credit/debit cards
- Payment processors (Stripe, etc.)
- Annual or monthly billing
- Automatic renewal with cancellation anytime

### Subscription Management

**Organization Admins Can:**
- View current plan and usage
- Upgrade/downgrade plans
- View billing history
- Download invoices
- Cancel subscription
- Add payment methods

### Grace Periods and Downgrade Handling

**When Subscription Lapses:**
- Grace period for payment issues
- Public forums become read-only (not deleted)
- Private capacity enforced (no new threads if over limit)
- Notification to admins
- Data retained for reasonable period
- Easy reactivation restores full access

**When Downgrading:**
- Content not deleted
- Access restrictions apply immediately
- Clear communication about what happens
- Option to archive or restructure before downgrade

## Non-Profit and Educational Discounts

**Eligible Organizations:**
- Registered non-profits
- Educational institutions
- Community organizations
- Open source projects
- Democratic organizing groups

**Discount Structure:**
- Verification required
- Significant discount on paid tiers
- Applied annually
- Transparent application process

## Pricing Philosophy

### Why This Model

**Sustainability:**
- Free tier builds user base
- Paid features fund development
- Revenue from organizations, not individuals
- Aligns incentives (better tools = more value = sustainable revenue)

**Accessibility:**
- Core functionality free ensures democratic access
- Paid features are "nice to have" not "must have"
- Small organizations can thrive on free tier
- Larger organizations pay to support infrastructure

**Fairness:**
- No artificial limits on free tier (generous capacity)
- Paid tiers unlock convenience and scale, not core features
- Pricing based on value delivered
- Non-profit discounts ensure mission alignment
